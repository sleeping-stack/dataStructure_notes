# 一道例题

~~~C
double f(int n, double a[], double x) {
    int i;
    double p = a[0];
    for (i = 1; i <= n; i++) {
        p += (a[i] * pow(x, i));
    }
    return p;
}

double f(int n, double a[], double x) {
    int i;
    double p = a[n];
    for (i = n; i > 0; i--) {
        p = a[i - 1] + x * p;
    }
    return p;
}
~~~

- 第一个算法时间复杂度为

$$
O(1)+O(2)+O(3)+\ldots+O(n)=O(n^2)
$$

- 第二个算法时间复杂度为$O(n^2)$
